# Common English Mistakes

Learn from common errors to improve your accuracy.

## Top Mistakes

### 1. Subject-Verb Agreement

**Incorrect**: He *don't* understand.
**Correct**: He *doesn't* understand.

### 2. Preposition Errors

**Incorrect**: Married *with*...
**Correct**: Married *to*...

### 3. Fewer vs. Less

**Fewer** is for countable nouns (fewer apples).
**Less** is for uncountable nouns (less water).

### 4. Who vs. Whom

Use **Who** as the subject (Who did it?).
Use **Whom** as the object (To whom did you send it?).

# The PERMISSION Codes (Modals)

!!! tip "The Operator Protocol"
    **Objective:** Modify the "mode" of the main verb to detect Probability, Ability, Or Obligation.

## 1. The Ability Matrix

* **Can:** Current Ability.
  * *Code:* "I **can** code in Python."
* **Could:** Past Ability.
  * *Code:* "I **could** run fast when I was young."
* **Be Able To:** General Capability (All Tenses).
  * *Code:* "I **will be able to** join you."

## 2. The Probability Matrix

* **Must:** 90-100% Certainty (Deduction).
  * *Code:* "He represents 100 clients. He **must** be busy."
* **Will:** 100% Future Certainty.
  * *Code:* "The sun **will** rise."
* **Should:** Expectation.
  * *Code:* "He left an hour ago. He **should** be here soon."
* **May / Might / Could:** 30-50% Possibility.
  * *Code:* "It **might** rain later." / "It **could** be true."
* **Can't:** 90-100% Negative Certainty (Impossibility).
  * *Code:* "That **can't** be true."

## 3. The Obligation Matrix

* **Must:** Internal Obligation / Written Rule.
  * *Code:* "I **must** call my mom." (I feel I need to).
* **Have To:** External Obligation.
  * *Code:* "I **have to** wear a tie." (Boss said so).
* **Should / Ought To:** Advice / Recommendation.
  * *Code:* "You **should** see a doctor."

## 4. The Interaction Matrix (Requests & Offers)

* **Can:** Informal Request.
  * *Code:* "**Can** I have some water?"
* **Could / May:** Formal Request.
  * *Code:* "**Could** you help me?" / "**May** I come in?"
* **Would:** Polite Offer/Invitation.
  * *Code:* "**Would** you like coffee?"

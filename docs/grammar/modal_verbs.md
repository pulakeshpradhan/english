# Modal Verbs

Modal verbs like *can*, *could*, *should*, *would*, *may*, and *might* express possibility, ability, permission, or obligation.

## Common Modal Verbs

| Modal | Meaning | Example |
| :--- | :--- | :--- |
| **Can** | Ability/Permission | I can swim. |
| **Could** | Past Ability/Suggestion | I could come if you want. |
| **May** | Possibility (Formal) | It may rain. |
| **Might** | Possibility (Less certain) | We might go. |
| **Must** | Obligation | You must finish this. |
| **Should** | Advice | You should see a doctor. |
| **Would** | Condition/Past Habit | I would be happy to help. |
